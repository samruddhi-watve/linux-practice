# 👤 User Management

Linux supports multiple users with different permission levels.

---

## 📚 Commands

| Command | Description |
|---------|-------------|
| `whoami` | Show current user |
| `id` | Display user ID |
| `groups` | Show user groups |
| `sudo` | Run command as administrator |
| `passwd` | Change password |

---

## 💻 Examples

```bash
whoami
id
groups
sudo apt update
passwd
```

---

## 🎯 Learning Outcome

- Identify current user
- Understand user privileges
- Use sudo safely
