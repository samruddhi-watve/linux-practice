# 🐚 Shell Scripting

Shell scripting helps automate repetitive Linux tasks.

---

## Topics

- Variables
- Input & Output
- Conditional Statements
- Loops
- Functions

---

## Example Script

```bash
#!/bin/bash

echo "Hello, Linux!"
```

---

## Running a Script

```bash
chmod +x script.sh
./script.sh
```

---

## 🎯 Learning Outcome

- Write basic Bash scripts
- Automate routine tasks
- Execute shell scripts
